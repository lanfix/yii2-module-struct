# Project Name

## Установка
Выполнить в корне проекта (скрипт копирования конфигов)
```
php build.php
```
Настроить подключение к БД
```
./config/db.php
```
Также настроить (можно закомментить дефайны)
```
./public/index.php
```
Установить зависимости композера
```
composer install
```
Применить миграции
```
php yii migrate
```

## Структура директорий
```
build/                      содержит файлы для сборки проекта
components/                 содержит общемодульные компоненты
    UserAuthComponent.php   компонент идентификации пользователя
config/                     содержит файлы глобальной конфигурации
    console.php             файл конфигурации консольного приложения
    web.php                 файл конфигурации web приложения
migrations/                 директория, содержащая файлы миграций БД
models/                     содержит общие модели (в основном всегда AR)
modules/                    содержит модули приложения
    admin/                  содержит файлы для админки
    frontend/               содержит файлы для фронтенда
    ...                     можно еще надобавлять
public/                     содержит файлы с публичным доступом (корень сайта)
runtime/                    содержит файлы, которые создаются yii во время работы
    cache/                  содержит какой-то кэш
    debug/                  содержит данные для debug-панели
    logs/                   содержит логи
vendor/                     зависимости компзера
```
